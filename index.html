What is smart-component?

Smart-component is a simple extension of the base HTMLElement class. It helps developers to easily create feature rich web components.

Features:

  • Unified attribute and property handling
  • Auto-bind property to state
  • Communication between parent and child components
  • Lighterhtml based templating
  • Local and global state management
  • Extra callbacks to never miss a moment

Templating

Options

listenChildren Subscribes to childrens which notifies parent.
notifyParent Notify parent on connect, disconnect and state changes.
watchContent Attach a mutation observer to the component and tunnel it to contentChangedCallback.
connectedChildren Stores a list of listened children.
render.container Default node to render content in.
render.template Default template to append to container.
render.globalState Call render on global state change.
render.autoAppendContainer Automatically append render container to component.

Callbacks

contentChangedCallback Called when any kind of change happens with the component or it's children.
childrenChangedCallback Called when child connects or any kind of change happens with the listened child's state.
renderCallback Called when rendering happens after state change. Optionally, you can listen on the full or partial global state too.

Helpers

_render() Force rerender of the component if needed for some reason.
_convertAttributeToBoolean(string) Makes string 'false' and empty to become boolean false.
_parseHTML(string) Creates real DOM element from string.
_dispatchEvent(eventName, [details={}], [bubbles=true]) Creates a custom event.
